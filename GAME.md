# connect.io

## Product summary

**connect.io** is a real-time 2-player word association game.

Two players join a match. In each round, both players have **15 seconds** to submit one word. The round resolves when:

- both players submit, **or**
- the 15-second timer expires.

Then both submitted words are revealed at the same time.

If the words match exactly according to the game rules, the game ends and the **score = round number** where the match happened. That score is saved and shown on the leaderboard.

Example:

- Round 1: `king` / `football` → no match
- Round 2: `messi` / `ronaldo` → no match
- Round 3: `fifa` / `fifa` → match → score = `3`

This game measures how well two people “connect.”
